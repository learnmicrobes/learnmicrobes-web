Badge — a small square or circular token for a letter, count, or status; the sage "A/B/C/D" answer marker, numeric step markers, and inline counts.

```jsx
<Badge variant="sage" size="lg">B</Badge>
<Badge variant="teal" shape="circle">3</Badge>
```

Variants: `sage`, `teal`, `soft`, `gold`, `error`, `onDark`. `shape="circle"` for step/count dots; default rounded square for answer letters.
