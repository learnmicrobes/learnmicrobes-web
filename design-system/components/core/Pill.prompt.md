Pill — the brand's fully-rounded tag; use for post types ("POP QUIZ"/"ANSWER"), difficulty ("INTERMEDIATE"), status, and URL chips.

```jsx
<Pill tone="dark">Pop quiz</Pill>
<Pill tone="sage">Intermediate</Pill>
<Pill tone="onDark" uppercase={false}>LearnMicrobes.com</Pill>
```

Tones: `dark` (deep teal), `teal` (brand), `sage` (light green), `soft` (sage tint), `outline` (hairline on white), `gold` (caution highlight), `onDark`. Set `dot` for a leading status dot; `uppercase={false}` for URLs / non-eyebrow text.
